# Exploratory Data Analysis Project by Python
## Topic: NYC Yellow Taxi Trips in December 2015
### Matplotlib and Pandas Implementation

Our goal is to clean the dataset, understand trip patterns, and extract meaningful insights from real-world transportation data given in our EEE 4416: Simulation Lab.
---
## Highlights and Key Insights:
* Cleaned data (removed zero/invalid fares, distances, coordinates, passengers)
* Converted datetime & extracted daily features
* Analyzed distributions (distance, fare, passengers etc.)
* Visualized trips, tips, tolls, and payment types
* Daily & weekly trip trends
* Revenue and passenger flow over the time
* Average trip duration per day
* Pickup & drop-off spatial patterns
* Simple traffic estimate (duration ÷ distance)
